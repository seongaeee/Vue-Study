- Multi Page App
- 여러 html으로 뷰 처리

#### 5. axios, vre-router, vue/cli

- axious, vre-router, vue/cli
```
1) axious
- 비동기 통신
- 하나의 화면을 a.js -> b.js 변경
- ajax 대신 이용

2) vre-router
- 웹 페이지 이동시 사용
- 컴포넌트 매핑
- controller처럼 url을 요청 처리

3) vue/cli
- vue.js를 사용해서 app을 만들때 필요한 기능 제공
- 파일을 쉽게 나누어줌
```
