### Vue Instance 관련 코드

1. this
- `methods`에서는 `this로 data 접근`

2. filter
- `{{ value | filter_name }}`
> 전역 필터
```javascript
 Vue.filter(
  "filter_name",
  (value, parameter) => {
    return "변환값";
  }
);
```
> 지역 필터
```javascript
new Vue({
  filters: {
    filter_name(value, parameter) {
      if (val.length == 0) 
        return "변환값";
    },
  },
});
```
